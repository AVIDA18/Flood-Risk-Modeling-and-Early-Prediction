On the RESOURCES folder there is Model train/Fit code in .pynb file, Model, Datasets and test-splitted data.

------------|
HOW TO RUN  |
------------|

FOR WINDOWS
------------

Go to https://www.python.org/ and download python version: 3.11.5
Open command prompt and get inside project directory
USE: pip install -r requirements.txt(it installs all the required libraries)
Python manage.py runserver (to run project)
GO TO: http://127.0.0.1:8000/
Enter Rainfall amount 
Enter Rever Level 
Click on Predict

FOR MACs
----------

Download and install Python 3.11.5 from https://www.python.org/ (macOS version).
Open Terminal.
Navigate to the project directory
pip3 install -r requirements.txt(it installs all the required libraries)
python3 manage.py runserver (to run project)
On the browser GOTO: http://127.0.0.1:8000/
Enter Rainfall amount 
Enter Rever Level 
Click on Predict
